Features

All disc versions

Original Episode I & II, the 1.1 revision, and Plus all connect. Same ship, cross-version play enabled.

Episode 3 ready

C.A.R.D. Revolution is fully supported — spectator teams, tournaments via the lobby battle table.

Server-side character slots

16 extra slots per account via $savechar / $loadchar. Move characters between versions.

Custom drop tables

Server-private drop mode keeps loot tamper-proof. Rare tables can be tuned per category.

Community quests

Beyond the disc's base quests — Ephinea, Schtserv, and GC-Forever community packs are installable.

Persistent games

$persist keeps a game alive after the last player leaves — drop in and out without losing your run.

Play

Pick your client

GameCube (Emulator)

Override one DNS line in Dolphin's config — desktop Dolphin (macOS, Windows, Linux) or Batocera handhelds/HTPCs. Same setting, just a different config path.

Desktop Dolphin →  ·  Batocera →

GameCube (Console)

Plug a Broadband Adapter into the GameCube's serial port. Set Primary DNS to 54.173.68.119 inside PSO's network setup screen. No emulator-side config involved.

Setup guide →

PC (Blue Burst)

The full PC version, pre-configured for this server — download, unzip, and play. The macOS build is a self-contained app; Windows is the classic client. You just need a registered UserID.

macOS Apple Silicon Windows 10 / 11
macOS — clear the download flag first. The app is unsigned, so macOS quarantines it. Open it before clearing that and you'll see "PSOBB is damaged… Move it to the Trash." Don't — click Cancel. Instead open Terminal, type the line below (keep the trailing space), drag the unzipped PSOBB-macOS folder onto the window, then press Enter — before you move the app to Applications:
xattr -drs com.apple.quarantine [ drag the PSOBB-macOS folder here ]
Don't type the path — the folder is often on the Desktop, not Downloads. (Windows is milder: SmartScreen → More info → Run anyway.) Full macOS steps →
