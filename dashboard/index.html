What this server supports

All disc versions

Original Episode I & II, the 1.1 revision, and Plus all connect. Same ship, cross-version play enabled.

Episode 3 ready

C.A.R.D. Revolution is fully supported — spectator teams, tournaments via the lobby battle table.

Server-side character slots

16 extra slots per account via $savechar / $loadchar. Move characters between versions.

Custom drop tables

Server-private drop mode keeps loot tamper-proof. Rare tables can be tuned per category.

Community quests

Beyond the disc's base quests — Ephinea, Schtserv, and GC-Forever community packs are installable.

Persistent games

$persist keeps a game alive after the last player leaves — drop in and out without losing your run.

Joining

Pick your client

First: send the admin your public IP from ifconfig.io. The server's DNS only answers for allowlisted addresses — without this, every other step will silently fail.

Dolphin (desktop)

One line in Dolphin.ini + a per-game config for the memory card. Works on macOS, Windows, and Linux.

Setup guide →

Batocera + Dolphin

For Batocera Linux handhelds or HTPCs running standalone Dolphin. Same setting, different config paths.

Setup guide →

Real GameCube + BBA

Plug a Broadband Adapter into the GameCube's serial port. Set Primary DNS to 54.173.68.119 inside PSO's network setup screen. No emulator-side config involved.

Dedicated guide coming.

PSO Blue Burst (PC)

The PC continuation of PSO. Different protocol and version family than the GameCube discs — uses a separate config block on the server side. Friends with BB clients (Tethealla, etc.) are welcome but the server-side setup is its own thing.

Setup guide coming.
